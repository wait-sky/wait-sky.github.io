# 后端技术

## Node.js 开发

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时，让 JavaScript 可以运行在服务器端。我主要使用以下 Node.js 相关技术：

- **Express**：轻量级的 Web 框架，用于构建 RESTful API
- **NestJS**：基于 TypeScript 的渐进式 Node.js 框架，提供完整的开发体验
- **Koa**：由 Express 原班人马打造的下一代 Web 框架
- **Fastify**：高性能的 Web 框架，专注于速度和低开销

## 数据库技术

数据库是后端开发的重要组成部分，我熟悉以下数据库技术：

- **关系型数据库**：MySQL、PostgreSQL、SQLite
- **NoSQL 数据库**：MongoDB、Redis、Cassandra
- **ORM 框架**：Sequelize、TypeORM、Prisma
- **数据库设计**：数据建模、索引优化、查询优化

## 微服务架构

微服务架构将应用拆分为多个独立的服务，每个服务专注于特定的业务功能：

- **服务发现**：Consul、Eureka、etcd
- **API 网关**：Kong、Nginx、Traefik
- **消息队列**：RabbitMQ、Kafka、Redis
- **容器化**：Docker、Kubernetes
- **服务监控**：Prometheus、Grafana、ELK Stack

## 云原生技术

云原生技术是现代后端开发的重要趋势：

- **Serverless**：AWS Lambda、Azure Functions、Google Cloud Functions
- **容器编排**：Kubernetes、Docker Swarm
- **CI/CD**：GitHub Actions、Jenkins、GitLab CI
- **基础设施即代码**：Terraform、CloudFormation
- **监控和可观测性**：OpenTelemetry、Jaeger
